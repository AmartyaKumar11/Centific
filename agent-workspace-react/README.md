# Agent Workspace React (Frontend-Only)

Minimal Next.js implementation of the agent-first loan case workspace.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Key Files

- `app/page.tsx` - full workspace UI + interactions
- `app/globals.css` - styling (clean, minimal, audit-friendly)
- `AGENT_COMPONENT_FUNCTIONALITY.md` - complete component and action spec for master agent consumption
